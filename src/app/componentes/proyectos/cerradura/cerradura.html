🔌 Cerradura Electrónica con Arduino

Este proyecto consistió en diseñar un sistema de seguridad con Arduino Uno que controla una cerradura electrónica. Se implementó un teclado matricial para ingresar una clave personal y un servomotor que se encarga de abrir o cerrar la cerradura.

El objetivo fue crear un dispositivo económico y accesible para mejorar la seguridad en el hogar. Aprendí sobre programación en C++, uso de librerías y conexión de periféricos electrónicos.
