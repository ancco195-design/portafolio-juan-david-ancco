📊 Sistema de Monitoreo de Pollos

El proyecto buscó mejorar la productividad en granjas rurales mediante un sistema de monitoreo automático de temperatura y humedad usando sensores conectados a un microcontrolador Arduino. Los datos se transmiten a una aplicación para llevar el control en tiempo real.

Gracias a este sistema, los granjeros pueden tomar decisiones más rápidas para mantener las condiciones óptimas de crianza. Aprendí a integrar sistemas embebidos, sensores y transmisión de datos.
